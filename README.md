# Programación Orientada a Objetos · CFGS DAW 2025/26

> Object-Oriented Programming coursework — *Programming* module, 1st year of the **Desarrollo de Aplicaciones Web** (DAW) higher vocational degree (Alicante, Spain).

This repository contains my OOP work in Java: in-class exercises, exams and a small study site. The focus is on solid object-oriented design — classes, encapsulation, inheritance and polymorphism — built on real Maven projects.

## 📚 What's covered

- **Object-oriented design** — classes, objects, encapsulation, inheritance and polymorphism.
- **Java + Maven** — every exercise/exam is a proper Maven project (`pom.xml`, `src/main/java`), not loose `.java` files.
- **Exams** — solved unit exams (`EXAMEN_UD1`, `EXAMEN_UD2`) kept as a record of progress.
- **Study site** — a small static site with the unit syllabus, published with GitHub Pages.
