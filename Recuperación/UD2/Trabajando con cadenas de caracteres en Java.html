1. Trabajando con cadenas de caracteres en Java (textos)

Clase String...

Desde el punto de vista de la programación diaria, uno de los tipos de datos más importantes de Java es String. String define y admite cadenas de caracteres.

En Java, los String son objetos. En realidad, has estado usando la clase String desde el comienzo del curso, pero no lo sabías. Cuando creamos un literal de cadena (entre comillas), en realidad estamos creando un objeto String. Por ejemplo, en la declaración:

System.out.println("En Java, los String son objetos");

El texto “En Java, los String son objetos” automáticamente se convierte en un objeto String, aunque no lo hayamos definido como tal.
