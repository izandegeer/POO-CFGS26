BONUS. ArrayList de objetos

El uso de vectores o arrays no tiene por qué restringirse a elementos de tipo primitivo. Por ejemplo, pueden crearse referencias a arrays de la clase Punto:

Pero en la POO no es común ceñirnos a los arrays tradicionales, sino usar otro tipo de listas "dopadas" para manipular mejor nuestros objetos. Estas listas son las llamadas ArrayList.

Un ArrayList permite almacenar elementos en memoria de manera dinámica. La principal diferencia con los arrays es que el número de elementos que almacena no está limitado por un número fijado al inicio.

La declaración de un ArrayList se hace según el siguiente formato:

ArrayList<nombreTipoClase> nombreDeLista = new ArrayList<>();

Entre < > indicamos la clase o tipos básicos de los objetos que se almacenarán:

ArrayList<String> listaPaises = new ArrayList<>();
Métodos típicos de los ArrayList
  • El método add posibilita añadir elementos, que se colocan después del último elemento que existiera en el ArrayList.

ArrayList<String> listaPaises = new ArrayList<>();
listaPaises.add("España"); //Ocupa la posición 0
listaPaises.add("Francia"); //Ocupa la posición 1
listaPaises.add("Portugal"); //Ocupa la posición 2

Es posible insertar un elemento en una determinada posición, desplazando el elemento que se encontraba en esa posición, y todos los siguientes, una posición más. Se utiliza el método add indicando como primer parámetro el número de la posición:

ArrayList<String> listaPaises = new ArrayList<>();
listaPaises.add("España");
listaPaises.add("Francia"); 
listaPaises.add("Portugal");
//El orden hasta ahora es: España, Francia, Portugal
listaPaises.add(1, "Italia");
//El orden ahora es: España, Italia, Francia, Portugal
  • El método get permite obtener el elemento almacenado en una determinada posición:

ArrayList<String> listaPaises = new ArrayList<>();
listaPaises.add("España");
listaPaises.add("Francia"); 
listaPaises.add("Portugal");
//El orden hasta ahora es: España, Francia, Portugal
listaPaises.add(1, "Italia");
//El orden ahora es: España, Italia, Francia, Portugal
System.out.println(listaPaises.get(3));
//Siguiendo el ejemplo anterior, mostraría: Portugal

Si tuviéramos un ArrayList con otro tipo de datos, como objetos que hayamos creado:

  • Para eliminar un determinado elemento se emplea el método remove al que se le puede indicar por parámetro un valor int con la posición a suprimir:

ArrayList<String> listaPaises = new ArrayList();
listaPaises.add("España");
listaPaises.add("Francia");
listaPaises.add("Portugal");
//listaPaises → España, Francia, Portugal
listaPaises.add(1, "Italia");
//listaPaises → España, Italia, Francia, 
Portugal
listaPaises.remove(2);
//listaPaises → España, Italia, Portugal
listaPaises.remove("Portugal");
//listaPaises → España, Italia

o bien, se puede especificar el elemento a eliminar si es encontrado en la lista:

  • El método indexOf retorna un valor int con la posición que ocupa el elemento que se indica por parámetro. Si no lo encuentra, devuelve -1.

ArrayList<String> listaPaises = new ArrayList();
listaPaises.add("España");
listaPaises.add("Francia");
listaPaises.add("Portugal");
//listaPaises → España, Francia, Portugal
listaPaises.add(1, "Italia");
//listaPaises → España, Italia, Francia, 
Portugal
listaPaises.remove(2);
//listaPaises → España, Italia, Portugal
listaPaises.remove("Portugal");
//listaPaises → España, Italia
String paisBuscado = "Francia";
int pos = listaPaises.indexOf(paisBuscado);
if(pos != -1) 
 System.out.println(paisBuscado + " encontrado en la posición: " + pos);
else System.out.println(paisBuscado + " no se ha encontrado");
  • Podemos recorrer un ArrayList con un bucle for como lo haríamos con los arrays convencionales. Para obtener el número de elementos, se utiliza el método size().

ArrayList<String> listaPaises = new ArrayList();
listaPaises.add("España");
listaPaises.add("Francia");
listaPaises.add("Portugal");
//listaPaises → España, Francia, Portugal
listaPaises.add(1, "Italia");
//listaPaises → España, Italia, Francia, Portugal

for(int i = 0; i < listaPaises.size(); i++) {
 System.out.println(listaPaises.get(i));
}

Como ya sabes, el foreach nos permite recorrer los elementos de un array sin utilizar un índice. Aunque la flexibilidad es menor que el bucle for normal, es muy útil para recorrer de forma típica un ArrayList.

  • set (posicion, nuevoValor) : modifica un elemento.
        ArrayList<String> listaPaises = new ArrayList();
        listaPaises.add("España");
        listaPaises.add("Francia");
        listaPaises.add("Portugal");
        listaPaises.add(1, "Italia");

        listaPaises.set(2,"Polonia");
        System.out.println(listaPaises.get(2));
  • clear() : borra todo el contenido de la lista.
        ArrayList<String> listaPaises = new ArrayList();
        listaPaises.add("España");
        listaPaises.add("Francia");
        listaPaises.add("Portugal");
        listaPaises.add(1, "Italia");

        listaPaises.clear();
        System.out.println("El tamaño de la lista es " + listaPaises.size()); //0
  • Object clone() : retorna una copia de la lista.
        ArrayList<String> listaPaises = new ArrayList();
        listaPaises.add("España");
        listaPaises.add("Francia");
        listaPaises.add("Portugal");
        listaPaises.add(1, "Italia");

        Object listaPaises_copia = listaPaises.clone();
        System.out.println(listaPaises_copia.toString());
  • boolean contains(Object o) : retorna true si se encuentra el elemento en la lista, y false en caso contrario.
        ArrayList<String> listaPaises = new ArrayList();
        listaPaises.add("España");
        listaPaises.add("Francia");
        listaPaises.add("Portugal");
        listaPaises.add(1, "Italia");


        if(listaPaises.contains("Suiza")){
            System.out.println("Tenemos a Suiza.");
        }else{
            System.out.println("No tenemos a Suiza.");
        }
  • boolean isEmpty() : retorna true si la lista está vacía.
        ArrayList<String> listaPaises = new ArrayList();
        listaPaises.add("España");
        listaPaises.add("Francia");
        listaPaises.add("Portugal");
        listaPaises.add(1, "Italia");

        //listaPaises.clear();

        if(listaPaises.isEmpty()){
            System.out.println("La lista está vacía");
        }else{
            System.out.println("La lista no está vacía");
        }
  • int lastIndexOf(Object o) : devuelve el índice de la última ocurrencia del elemento. Si el elemento no se encuentra, devuelve -1.
        ArrayList<String> listaPaises = new ArrayList();
        listaPaises.add("España");
        listaPaises.add("Francia");
        listaPaises.add("Portugal");
        listaPaises.add(1, "Italia");

        System.out.println(listaPaises.lastIndexOf("Francia")); //2
        System.out.println(listaPaises.lastIndexOf("Suiza")); //-1
  • addAll(Arrays.asList(elemento1, elemento2, elemento3,...)) : añadir más de un elemento a la lista.
        ArrayList<String> listaPaises = new ArrayList();
        listaPaises.add("España");
        listaPaises.add("Francia");
        listaPaises.add("Portugal");
        listaPaises.add(1, "Italia");

        listaPaises.addAll(Arrays.asList("Polonia","Suiza","Alemania"));
        System.out.println(listaPaises.toString());

También se puede hacer desde un método a parte, usando varargs: 

public class Main {
    public static void main(String[] args) {

        ArrayList<String> listaPaises = new ArrayList();
        listaPaises.add("España");
        listaPaises.add("Francia");
        listaPaises.add("Portugal");
        listaPaises.add(1, "Italia");

        agregarVariosPaises(listaPaises,"Polonia","Suiza","Alemania");
        System.out.println(listaPaises.toString());
        
    }

    public static void agregarVariosPaises(ArrayList<String> lista, String... paises) {
        lista.addAll(Arrays.asList(paises));
    }

}
