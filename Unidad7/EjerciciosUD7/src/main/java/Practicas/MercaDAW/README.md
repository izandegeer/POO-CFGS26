# MercaDAW

Simulacion de compra online en un supermercado por consola.

## Clases

- **Producto** - Enum con los productos disponibles y sus precios.
- **Pedido** - Carrito de la compra con un HashMap de productos y cantidades. Incluye promociones 3x2 y 10% de descuento.
- **Cliente** - Usuario con credenciales, direccion y su pedido asociado.
- **MercaDAW** - Genera clientes aleatorios con usuario y contrasenya random.
- **AppZonaCliente** - Aplicacion principal con login, seleccion de productos, resumen del carrito y aplicacion de promos.
